# Amélioration de l'UX la page détail projet

## Contexte

Actuellement, on entre dans le détail de la page d'un projet par la route "/orgs/mapado/projects/desk".

Seulement cette page n'a pas un intérêt fondamental pour l'utilisateur.

## Objectif

Le but de ce changement est d'avoir la page d'accueil d'un projet qui soit la liste des traductions.
On va utiliser un système de navigatation dans le projet avec trois pages :

- Liste des traductions
- paramérage du projet (langues, etc.)
- import et export des traductions
