# Cas spécifique en cas d'une seule organisation.

## Contexte

Actuellement, On ne peux avoir qu'un user par organisation

## Besoin

Besoin de pouvoir ajouter des utilisateurs a une organisation.

## Specification

- Ajouter une page de gestion des utilisateurs dans les paramètres de l'organisation
- Permettre d'inviter un utilisateur
- Permettre de lister les utilisateurs de l'organisation
- Permettre de supprimer un utilisateur de l'organisation

On ne vas pas gérer l'envoi de mail pour le moment, l'invitation sera juste un lien dans l'interface dans la liste des utilisateurs.
Un code d'invitation permet à un utilisateur invité de rejoindre l'organisation, que ce soit un nouveau compte ou un compte existant.

## Contraintes

Tu dois toujours lire le fichier README.md et les décisions techniques précédentes.
